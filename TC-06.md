**Description:** Verify the validation of the "Source" field (optional, must be a valid URL).

**Preconditions:**

- The user is logged into the system.
- The "Create News" form is open.

**Steps:**

1. Navigate to [GreenCity News](https://www.greencity.cx.ua/#/greenCity/news) and click "Create News".
2. Leave the "Source" field empty.
3. Fill in all mandatory fields and click "Publish".
4. Verify that the news is published successfully without errors.

1. Enter an invalid URL in the "Source" field (e.g., "www.example.com").
2. Verify that:

- The "Publish" button remains disabled.
- An error message appears in red:
- "Please add the link of the original article/news/post. The link must start with http(s)://".

1. Enter a valid URL in the "Source" field (e.g., "https://example.com").
2. Verify that:

- The error message disappears.
- The "Publish" button becomes enabled.
- Click "Publish".
- Verify that the news is published successfully with the provided source link.

**Expected Result:**

- An empty "Source" field is allowed.
- An invalid URL triggers an error message in red, and the "Publish" button remains disabled.
- A valid URL is accepted, and the "Publish" button becomes enabled.

**Automated Verification:**

- Check if the error message appears for an invalid URL.
- Verify that the "Publish" button remains disabled until a valid URL is entered.
- Confirm that news is published successfully with a valid source.
