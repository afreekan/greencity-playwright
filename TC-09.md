**Description:**
Verify that only the author of the news can see the "Edit news" button.

**Preconditions:**

- User is logged in.
- User has previously created a news post.

**Steps:**

- Log in as the user who created the news.
- Navigate to the news post.
- Check for the presence of the "Edit news" button.

**Expected Result:**

- The "Edit news" button is visible to the author.
- Automated Verification:
- Assert that the "Edit news" button is displayed on the page.
